## Контекст

СТТ (ООО «Современные технологии торговли») — аттестованный EDI-провайдер в Беларуси, резидент ПВТ. На рынке с 2010 года, 40 млн переданных EDI-сообщений с 2011 года, 7000+ активных юрлиц и ИП в системе, 1000+ интеграций, микросервисная архитектура OpenAPI.

Материал подготовлен как рабочий инструмент для приоритизации продуктовой стратегии — не официальная позиция компании.

## Обновление контента

Файл `index.html` — самодостаточный (HTML/CSS/JS без сборки). Чтобы обновить данные, отредактируйте массив `services` в `<script>` и загрузите файл заново в репозиторий — GitHub Pages пересоберётся автоматически.
